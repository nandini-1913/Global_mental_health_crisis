# 🌍 Global Mental Health Analysis

## 📊 Project Overview
This project analyzes global mental health trends focusing on:
- Depression (%)
- Anxiety (%)

## 🇮🇳 India Insights
- Compared India's mental health statistics with global averages
- Identified trends in depression and anxiety levels

## 📈 Visualizations
- Country vs Depression (%)
- Country vs Anxiety (%)
- India vs Global Comparison

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 🎯 Key Insights
- Mental health issues are significant globally
- India shows notable trends due to population and socio-economic factors

## 🚀 Conclusion
Mental health awareness and accessibility to healthcare need improvement, especially in developing countries.
